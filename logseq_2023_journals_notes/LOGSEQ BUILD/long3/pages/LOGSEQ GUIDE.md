- ## Basic Shortcuts
	- Create New Block - **ENTER**
	- New Line in Current Block - **SHIFT + ENTER**
	- Indent Block - **TAB**
	- Outdent Block - **SHIFT + TAB**
	- Select All Blocks - **CTRL + SHIFT + A**
	- Full Text Search - **CTRL + K**
	- Search in the Current Page - **CTRL + SHIFT + K**
- ## Triggers
	- Slash Autocomplete - **/**
	- Block Content Autocomplete - **<**
	- Page Reference Autocomplete **[[]]**
	- Block Reference - **(())**
	- Open Link in Sidebar - **SHIFT + CLICK**
	- Block Context Menu - **RIGHT CLICK**
