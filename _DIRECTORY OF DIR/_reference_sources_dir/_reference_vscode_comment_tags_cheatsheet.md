### Better Comments Tags Cheatsheet

- `!`
- `added`
- `edited`
- `refactor`
- `removed`
- `-`
- `?`
- `//`
- `todo`
- `*`
- `fixme`
- `note`
- `reference`
- `fixed`

- remember to add three colons to preface the tag ':::' IE: ':::example'

//fixme: This section contains a bug that needs fixing


This is a note admonition.
