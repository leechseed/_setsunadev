## 7. Aspect Logic Gates (Computational Conflict)

Geometric relationships between variables function as Boolean logic gates.

- **CONJUNCTION (AND Gate):** Two variables fuse into a single command. (e.g., Sun conjunct Mars = "To exist is to act.")
    
- **SQUARE (NOT Gate):** Friction that forces a state change. Variable A prevents Variable B from executing, requiring a **Systemic Turning Point**.
    
- **OPPOSITION (XOR Gate):** A binary choice. The Character cannot execute both simultaneously (e.g., Logic vs. Desire).
    
- **TRINE (OR Gate):** A "Passive Talent." Data flows freely between points, often leading to **Systemic Stagnation** because there is no friction to force growth.
